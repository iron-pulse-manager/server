## 🎯 프로젝트 개요

### **목표**
- **출시일**: 2024년 10월 31일 (16주간 개발)
- **팀 구성**: 2명 (웹 담당 + 앱 담당)
- **MVP 범위**: 헬스장 운영 + 회원/직원 앱 핵심 기능

### **플랫폼별 기능 분담**
| 플랫폼 | 주요 사용자 | 핵심 기능 |
|--------|------------|-----------|
| **웹** | 사장님 | 회원관리, 매출관리, 상품관리, 직원관리, 통계, 락커관리, 일일권관리, 지출관리 |
| **앱 (직원용)** | 직원 | 담당회원 조회, 운동일지 작성, 식단관리, 레슨관리 |
| **앱 (회원용)** | 회원 | 레슨일정, 운동일지 조회, 식단기록, 체중관리 |

### **MVP 필수 기능 (10월 출시)**
| 우선순위 | 기능 | 플랫폼 |
|---------|------|--------|
| **P0 (필수)** | 사용자 관리 (Owner, Employee, Member) | 웹 + 앱 |
| **P0 (필수)** | 사업장 관리, 상품 관리 | 웹 |
| **P0 (필수)** | 결제 및 이용권 관리 | 웹 |
| **P0 (필수)** | 회원 관리 (상세정보, 검색, 상태관리) | 웹 |
| **P0 (필수)** | 락커룸 관리, 일일권 관리 | 웹 |
| **P0 (필수)** | 지출 관리, 매출 통계 | 웹 |
| **P0 (필수)** | 운동일지 관리 | 앱 |
| **P0 (필수)** | 식단 관리 | 앱 |
| **P0 (필수)** | 레슨 관리 | 웹 + 앱 |

## 👥 팀 구성 및 역할 분담

### **개발자 A (웹 담당 + Tech Lead)**
- **주요 책임**: 웹 기능 전담, 아키텍처 설계, 배포 관리
- **담당 기능**:
  - 사장님 웹사이트 모든 기능
  - Owner, Business, Product, Payment 도메인
  - Locker, DayPass, Expense, Statistics 도메인
  - 권한 관리 및 인증 시스템
- **부가 업무**: CI/CD 설정, 데이터베이스 관리, 기술 의사결정

### **개발자 B (앱 담당)**
- **주요 책임**: 앱 기능 전담, API 개발, 모바일 최적화
- **담당 기능**:
  - 직원 앱/회원 앱 모든 기능
  - Employee, Member 도메인
  - Workout, Diet 도메인 
  - Schedule (레슨 관리) 도메인
  - OAuth 인증 (카카오/애플)
- **부가 업무**: API 문서화, 모바일 성능 최적화, 테스트 작성

### **공통 책임**
- 매일 스탠드업 미팅 (15분)
- 주간 스프린트 리뷰 및 계획
- 코드 리뷰 (모든 PR 2명 리뷰)
- 도메인 간 연동 및 API 설계

## 🎯 MVP 기능 범위

### **웹 기능 (사장님용)**

#### **1. 핵심 관리 기능**
- [x] **로그인 및 권한 관리**
  - 사장님 로그인 (ID/PW)
  - 마스터/일반 권한 모드 전환
  - 비밀번호 찾기

- [x] **대시보드**
  - 마스터 권한: 일/월 매출, 방문자 수, 매출 구성비, 방문자 추이
  - 일반 권한: 회원 수, 만료임박 회원, 월별 캘린더

- [x] **회원 관리**
  - 회원 목록 조회 (활성/만료임박/만료 상태별)
  - 회원 상세 정보 (기본정보, 이용권, 출석, 결제내역)
  - 회원 검색 및 필터링
  - 이용권 등록/수정/삭제
  - 회원 정보 수정

#### **2. 운영 관리 기능**
- [x] **상품 관리**
  - 상품 등록/수정/삭제 (헬스, 개인레슨, 락커, 기타)
  - 상품 활성화/비활성화

- [x] **락커룸 관리**
  - 200개 락커 현황 조회
  - 락커 배정/해제
  - 락커 상태 관리 (사용중, 만료임박, 만료, 비어있음)

- [x] **일일권 관리**
  - 일일권 등록 (이름, 연락처, 방문일시, 결제정보)
  - 일일권 목록 조회 및 검색
  - 일일권 통계

- [x] **직원 관리**
  - 직원 목록 조회
  - 직원 가입 승인/거절
  - 직원 상세 정보 및 실적 조회
  - 담당 회원 관리

#### **3. 매출 및 분석**
- [x] **지출 관리**
  - 지출 등록 (인건비, 임대료, 수도/경비, 기타)
  - 월별 캘린더 지출 현황
  - 지출 통계

- [x] **통계**
  - 회원 통계 (상태별, 월별 현황, 유형 분포)
  - 매출 통계 (월별 매출, 수입 요약, 지출 요약)
  - 직원 실적 통계

### **앱 기능**

#### **직원 앱**
- [x] **홈 화면**
  - 이번달 나의 매출 조회
  - 주별 일정 조회 및 등록

- [x] **회원 목록**
  - 담당 회원 목록 조회
  - 회원 상세 조회 (전화걸기, 메시지 발송)
  - 운동일지 조회
  - 식단기록 조회

- [x] **운동일지 관리**
  - 운동일지 대상 조회
  - 운동일지 작성
  - 회원 질문에 대한 답변

- [x] **식단 피드백**
  - 회원별 식단 지정
  - 회원 식단 기록 조회 및 피드백

- [x] **레슨 관리**
  - 회원이 신청한 레슨 목록 조회
  - 레슨 수락/거절

#### **회원 앱**
- [x] **홈**
  - 목표 몸무게, 현재 몸무게 표시

- [x] **개인 레슨 일정**
  - 월별 일정 조회
  - 레슨 신청

- [x] **운동일지 조회**
  - 날짜별 운동일지 조회
  - 운동 질문 작성 및 직원 답변 확인

- [x] **식단 등록**
  - 일별 식단 목록 조회 (직원 추천 식단)
  - 일별 식단 기록
